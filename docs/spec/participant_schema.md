# Specification: Participant Data Schema

Official contract between the Mobile Frontend and Python Backend Worker (FastAPI).

## Collection Path
`rooms/{roomId}/participants/{participantId}`

## Document Fields

| Field Name | Type | Description | Created By |
| :--- | :--- | :--- | :--- |
| `username` | `string` | User's display name | Frontend |
| `pronoun` | `string` | User's preferred pronouns | Frontend |
| `ocean_vector` | `number[]` | 5-dimensional vector `[V0..V4]`. Values: `0-100`. | Frontend |
| `swipeAnswers` | `number[]` | 10 binary choices (`0` or `1`) from the Soul Test. | Frontend |
| `mood` | `string` | Selected mood emoji/label (if `inputMode` is `mood`) | Frontend |
| `imageUrl` | `string` | Firebase Storage URL (if `inputMode` is `camera`) | Frontend |
| `inputMode` | `string` | `mood` or `camera` | Frontend |
| `language` | `string` | `en`, `jp`, or `cn` | Frontend |
| `status` | `string` | `generating_questions` → `answering` → `waiting_for_ai` → `ready` | Shared |
| `qa` | `array` | List of `{ questionId, question, answer }` objects. | Frontend |
| `questions` | `array` | List of `{ id, text }` generated by AI. | Backend |
| `aiTopics` | `array` | 3 trilingual topic objects `{ en, cn, jp }`. | Backend |
| `avatarUrl` | `string` | Generated avatar URL. | Backend |
| `animal_spirit` | `string` | AI-generated spirit animal description. | Backend |
| `final_vector` | `number[]` | Refined 5D vector after processing answers. | Backend |
| `tags` | `array` | 2 trilingual tag objects `{ en, cn, jp }`. | Backend |
| `createdAt` | `timestamp` | Server side timestamp | Frontend |
| `expiresAt` | `timestamp` | Expiration (usually 2hrs after creation) | Frontend |

## State Machine
1.  **Frontend**: Sets status to `generating_questions`.
2.  **Backend**: Listens for `generating_questions`, generates `questions`, updates status to `answering`.
3.  **Frontend**: User submits answers, sets status to `waiting_for_ai`.
4.  **Backend**: Listens for `waiting_for_ai`, generates profile/matches, updates status to `ready`.
